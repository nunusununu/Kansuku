# 艦すく

ダウンロードは右側にある「Release」から行ってください。常に最新バージョンをご利用ください。






## 概要

このソフトウェアを使用すると、簡単にスクリーンショットを撮ることができます。

タスクトレイ常駐型のため、タスクバーを占有しません。

優れたショートカットのおかげで画面を妨げることなくスクリーンショットを撮ることができます。
